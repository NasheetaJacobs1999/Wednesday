There are different programming languages that involve web development

HTML

Hypertext Markup Language, or HTML, is a programming language used to describe the structure of web pages. HTML makes it possible to create static pages with text, headings, tables, lists, images, links, and so on. HTML is the "bonestructure" of a webpage.

JavaScript

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, etc. Eg. The interactivity that happens when you click a button or type in a box

CSS

CSS is responsible for the text style, size, positioning, color, and more on a website. It's also what controls how a website's style shifts between desktop and mobile versions. CSS is the coding language that makes your website look beautiful and full of colour.
